Tip #1: Optimizing Images for the Web

tinyjpg redactor allows uploaded image to be analyzed and the best possible JPEG encoding is applied. Based on the content of your image an optimal strategy is chosen. The result is a quality image without wasting storage or bandwidth!

I optimized my origian profile image by using the online service tinyjpg redactor which reduced my image by 57% and reduced the size from 561KB to 242KB.
